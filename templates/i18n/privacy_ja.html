本ポリシーは、Punk Code Solutionが運営するFinanças Newsが、サイト利用時に個人データをどのように取り扱うかを説明します。処理はブラジルの一般データ保護法（LGPD・法律第13.709/2018号）およびその他の適用法令に従います。

1. 取り扱う可能性のあるデータ

  • ニュースレター: 任意で提供されたメールアドレスと登録日;
  • 技術データ: IPアドレス、ブラウザ、端末、閲覧ページ、日時、セキュリティログ;
  • ローカル設定: 明暗テーマ、言語、閲覧設定（localStorage／Cookie）;
  • 広告・計測: 有効化された提供者による識別子、Cookie、操作データ。

ニュース閲覧のために金融口座情報、身分証、銀行パスワードを求めることはありません。

2. 目的と法的根拠

データは次の目的で使用される場合があります:

  • ニュースレター配信と登録管理;
  • 設定、性能、ポータルのセキュリティ維持;
  • 不正利用・詐欺・不正アクセスの防止;
  • 利用統計の作成および（有効時の）広告表示;
  • 法的義務の履行および権利行使への対応。

3. Cookieとローカル保存

界面と言語の設定を記憶するためにローカル保存とCookieを使用します。有効時にはGoogle AdSenseなどの第三者が広告配信・計測のためにCookieを使用する場合があります。

4. 共有と処理者

ホスティング、データベース、ニュースレター、セキュリティ、広告など運営に必要な提供者によって処理されることがあります。メーリングリストや個人データの販売は行いません。

5. 外部・アフィリエイトリンク

報道機関・出典・パートナーへのリンクは、それぞれの第三者ポリシーに従います。

6. 国際転送

一部の提供者はブラジル国外でデータを処理する場合があり、適用されるLGPDに適合する保護措置を求めます。

7. 保存とセキュリティ

記載目的に必要な期間データを保持し、合理的な技術的・組織的セキュリティ対策を講じます。

8. あなたの権利

LGPDに基づき、確認・アクセス、訂正、匿名化・遮断・ポータビリティ・削除、共有に関する情報、同意の撤回などを請求できる場合があります。

9. 児童・青少年

児童のデータを意図的に収集しません。保護者は確認と削除を請求できます。

10. 更新と問い合わせ

本ポリシーは更新される場合があります。権利行使やプライバシーに関する質問は、Punk Code Solutionの公式窓口をご利用ください。
